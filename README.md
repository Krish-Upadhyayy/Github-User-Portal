# GitHub User Explorer

A web application that allows users to search GitHub profiles and explore developer information, repositories, and programming language usage using the GitHub REST API.

##  Features

- Search GitHub users by username
- View developer profile information
- Display profile avatar, bio, and location
- View followers and following count
- Display public repository count
- Access GitHub profile through a direct link
- Browse user repositories
- Sort repositories by:
  - Stars
  - Recently Updated
  - Name
- View language distribution across repositories
- Loading and error handling states

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- GitHub REST API
- Fetch API
- Async/Await
